Инструкция к программе Project10B.py
1) Скачать файлы (скопировать код): Project10B.py, Face.xml
2) Открыть файлы редакторе кода (советую Idle)
3) Установить библиотеки mediapipe и cv2 (pip install mediapipe, pip install opencv-python)
4) Запустить программу
ВАЖНО: В программе используется библиотека mediapipe и cv2, для работы обеих этих библиотек нужна версия python 3.11. Файлы пометить в одну папку
